화면 동작 요약

• 전화 들어오면 화면 상단 중앙에 navy(검정 가까운 남색) 배너가 위에서 부드럽게 슬라이드 다운
• 배너는 어떤 탭에 있든(테이블/주문/주문현황/관리자) 무조건 위에 떠 있음 — 작업 안 끊김
• 주문받기 클릭 → 자동으로 "배달" 탭으로 이동 + 빈 배달 슬롯(d1, d2 …)에 주소 자동 채움 → 사장님은 메뉴만 누르면 됨
• ✕ 클릭 → 배너만 사라짐(이미 받은 전화 다시 안 떠)
• 같은 번호로 다시 걸려오면 같은 배너가 다시 표시 + 주문횟수 갱신
