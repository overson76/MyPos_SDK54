### 풀세트 첫 적용에서 배운 것

- **사고 #1**: 첫 sync 시 메인 `.env` mtime (5/7) > NAS `.env` mtime (5/2) → 메인 → NAS 방향 push → NAS 의 카카오 키 32자 (옛 채워진 값) 가 메인의 빈 값으로 덮어씌워짐. **사장님이 NAS 복구 후 정상화**. 교훈: sync 가 mtime 함정 — NAS 를 source of truth 로 취급해야 안전.
- **사고 #2**: macOS .dmg 빌드 실패 (`_mkfifoat` symbol not found) — macOS 12.6 의 dyld 가 dmg-builder@1.2.0 의 python3.14 와 비호환. .zip 만 업로드됨. 매장 운영 영향 0 (.exe 가 본질).
- **빌드 플래그**: `electron-builder --publish always` 만으로는 host (macOS) 만 빌드. Windows .exe 받으려면 **`--win` 명시 필수**.
- **워크트리에 node_modules 없음** → electron-builder 는 반드시 메인 리포에서 실행. 워크트리는 코드 작업 전용.
- **EAS CLI 미설치** — `npx -y eas-cli@latest` 로 우회 (NPM_CONFIG_CACHE=/tmp 권한 우회 필요).
